# AI × Web3 School 个人学习计划

> 基于 Handbook 四层地图，结合个人画像定制。
>
> 画像：AI 新手 / Web3 有基础 / 编程基础脚本 / 目标开发方向
> 每日投入：4–6 小时

---

## 第一阶段：AI 基础入门（预计 2–3 周）

**目标**：建立 AI / LLM / Agent 的核心概念框架，能理解并上手使用。

| 周次 | 主题 | Handbook 章节 | 实践 |
|------|------|---------------|------|
| 1 | 大语言模型（LLM） | [LLM](https://aiweb3.school/zh/handbook/ai-basics/llm/) | 使用 Claude Code / Hermes 跑通一个基础对话流程 |
| 1–2 | 提示词（Prompt） | [Prompt](https://aiweb3.school/zh/handbook/ai-basics/prompt/) | 练习写 prompt，对比不同 prompt 效果 |
| 2 | 上下文（Context） | [Context](https://aiweb3.school/zh/handbook/ai-basics/context/) | 理解 token、窗口、长上下文 |
| 2 | 检索增强生成（RAG） | [RAG](https://aiweb3.school/zh/handbook/ai-basics/rag/) | 了解 RAG 原理 |
| 2–3 | 智能体（Agent） | [Agent](https://aiweb3.school/zh/handbook/ai-basics/agent/) | 尝试用 Hermes 写一个简单 agent 工作流 |
| 3 | 框架 / Vibe Coding / MCP | 相关章节 | 结合 Web3 背景，探索 MCP |

## 第二阶段：Web3 基础补强（穿插进行）

**目标**：补齐 DeFi 等知识短板，温故知新。

| 主题 | 备注 |
|------|------|
| DeFi | 新的领域，重点学习 |
| 账户抽象 | 已有基础，快速过 |
| 安全 | 结合合约经验深入 |
| 其他 Web3 章节 | 按需阅读 |

## 第三阶段：AI × Web3 Bridge（预计 3–4 周）

**目标**：真正打通 AI 和 Web3，能做出可运行的最小产品。

| 主题 | 说明 |
|------|------|
| 链感知上下文 | Agent 如何理解链上状态 |
| Web3 工具调用 | Agent 调用合约、查询链上数据 |
| 智能体工作流 | 自动化流程设计 |
| 智能体钱包 | Agent 控制钱包 |
| 机器支付 / 结算与托管 | 自动化支付场景 |
| 智能体身份 / 信任与声誉 | 去中心化身份 |
| AI 预言机 / 可验证 AI | 链上 AI 验证 |
| AI 安全 / 隐私 / 主权 | 安全边界 |

## 第四阶段：前沿探索 + Hackathon 项目

**目标**：把学到的组合成一个完整的 AI × Web3 产品。

| 主题 | 说明 |
|------|------|
| 智能体商业 | 商业模式探索 |
| 开发工具 | 实用工具链 |
| 钱包与权限 | 权限模型 |
| AI 安全 / 治理 | 更深层话题 |

---

## 学习方式

- 每天先读 WCB Learning 页面确认当日任务
- 读 Handbook 对应章节 → 写 daily note
- 动手练习：用 Hermes / Claude Code 做小实验
- 问题沉淀到 handbook-feedback/
- 每周末复盘，调整下周计划
